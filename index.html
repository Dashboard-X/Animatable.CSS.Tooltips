More tooltips…

A short time ago I made some pure CSS3 Tooltips and while these are a clean and simple way of adding tooltips, the lack of browser support for animations applied to pseudo-elements makes them a little boring.

These tooltips require a little extra markup, but it's well worth it

View CSS

Base markup

The base markup for these tooltips is pretty straightforward. The content for your tooltip goes between the span tags.

<a href="#" class="tip-trigger"><span>Yay Tooltip!</span>hover</a>

Adding animations

The interesting stuff starts when you add CSS3 animations into the mix. This is done simply by adding another class to the anchor tag, for example:

<a href="#" class="tip-trigger tipFade"><span>Yay Tooltip!</a>

class="tipFade" - Yay Tooltip!hover over me.

class="tipFadeDown" - Yay Tooltip!hover over me.

class="tipFadeUp" - Yay Tooltip!hover over me.

class="tipFadeLeft" - Yay Tooltip!hover over me.

class="tipFadeRight" - Yay Tooltip!hover over me.

class="tipBounceDown" - Yay Tooltip!hover over me.

class="tipZoomIn" - Yay Tooltip!hover over me.

class="tipZoomOut" - Yay Tooltip!hover over me.
